### 小程序项目结构

- pages目录：存放小程序页面文件
- utils目录：存放工具模块
- app.js：程序入口
- app.json：全局配置文件
- app.wxss：全局样式配置文件
- project.config.json：项目配置文件
- sitemap.json：索引配置文件（配置小程序及其页面是否允许被微信索引）

### 小程序页面组成

小程序页面在pages目录中以文件夹的形式存在，其中包含四个文件：

- .js文件：页面的脚本文件，存放页面数据，事件处理方法等
- .json文件：页面配置文件
- .wxml文件：页面模板结构文件
- .wxss文件：页面样式表文件

```bash
+---example		 			# 页面文件夹
|       example.js
|       example.json
|       example.wxml
|       example.wxss
```

### 添加页面

在app.json的pages标签中直接添加新页面的路径，保存后就会自动生成新页面的文件

```json
{
  "pages": [
    "pages/index/index",
    "pages/logs/logs",
    "pages/example/example"		// 这是新增加的页面
  ]
}
```

### 修改首页

修改app.json的pages标签中各个标签的顺序，处于第一个的页面就是首页

```json
{
  "pages": [
    "pages/index/index",		// index页面为首页
    "pages/logs/logs",
    "pages/example/example"
  ]
}
```
